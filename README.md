# Кубятня 2.0

Обновлённая версия VK-бота для бросков кубов, цитат и автоответов.

## Что изменилось

- Код разделён на модули.
- Поддерживаются положительные и отрицательные модификаторы:
  - `[3к20+5`
  - `[3к20-5`
- Несколько бросков в одном сообщении выводятся компактным списком.
- На всё сообщение выдаётся одна цитата.
- Цитаты не повторяются, пока не закончится весь пул.
- Триггеры вынесены в `data/replies.json`.
- КД каждого триггера считается отдельно для каждой беседы.
- КД равен 20 минутам.
- Картинка загружается только при первом срабатывании, затем используется сохранённый attachment.

## Установка

1. Распакуйте архив.
2. Скопируйте содержимое в корень существующего репозитория.
3. Не удаляйте текущие PNG-файлы из корня репозитория.
4. Убедитесь, что Render запускает:

```text
python bot.py
```

5. Запустите сервис.

## Постоянные attachment

После первой успешной загрузки бот записывает attachment в:

```text
data/attachments.json
```

На Render этот файл может сброситься после нового развёртывания. Чтобы исключить повторную загрузку полностью, можно вручную перенести значения из него в поля `attachment` файла `data/replies.json`.

## Формат цитат

Старый вариант — одна цитата на строку.

Для многострочных цитат используйте разделитель:

```text
Первая цитата
===
Первая строка второй цитаты
Вторая строка второй цитаты
===
Третья цитата
```
